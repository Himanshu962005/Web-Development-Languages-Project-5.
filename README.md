# 1.Project - Netflix Clone. 🎬
A modern and responsive Netflix landing page clone built using HTML and CSS. This project recreates the official Netflix homepage UI with a clean layout, smooth design structure, and mobile-friendly responsiveness.

## 🚀 Features
* Responsive Navigation Bar.
* Hero Section with Background Overlay.
* Email Signup Section.
* Feature Showcase Sections.
* FAQ Section.
* Footer with Multiple Links.
* Hover Effects and Smooth UI.
* Mobile + Tablet Responsive Design.
* Flexbox and CSS Grid Layout.

## 🛠️ Tech Stack
* HTML5.
* CSS3.
* Flexbox.
* CSS Grid.
* Media Queries.

## 📚 What I Learned
This project helped me improve my frontend development skills by practicing:
* Real-world website structuring.
* Responsive web design.
* Clean UI development.
* Layout building using Flexbox and Grid.
* Professional project organization.

## 🎯 Project Goal
The goal of this project was to strengthen my frontend foundation before moving to JavaScript and React development.
